Reserved Unicode Characters

This section provides a short list of built-in constants, functions, and operators in Derive with their Unicode hexadecimal value, character symbol, name, comments and methods of input.

Constants

E7C2        ð        area of unit circle (not the same as the Greek letter ð)
                       pi, Ctrl+Backspace E7C2, Ctrl+P, Math symbol toolbar

E7C0        e        base of natural logarithms (not the same as the Latin letter e)
                       #e, Ctrl+Backspace E7C0, Ctrl+E, Math symbol toolbar

E7C1        i        imaginary unit (not the same as the Latin letter i)
                       #i,  Ctrl+Backspace E7C1, Ctrl+I, Math symbol toolbar

E7C3        ã        Euler’s gamma constant (not the same as the Greek letter ã)
                       euler_gamma, Ctrl+Backspace E7C3, Math symbol toolbar

221E        ∞        positive infinity
                       inf, Ctrl+Backspace 221E, Ctrl+0, Math symbol toolbar
